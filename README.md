# 各項稻作規模別排序全部鄉鎮資料讀取
從農業部農糧署資料庫中讀取各項稻作規模別排序全部鄉鎮資料，轉換成.csv 檔案

資料將從農業部農糧署資料庫[https://agr.afa.gov.tw/afa/afa_frame.jsp](https://agr.afa.gov.tw/afa/afa_frame.jsp)讀取
